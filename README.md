# Puppygirl-Linux
A Linux distro (based on arch btw) For puppygirl's, By a puppygirl! :33

## What is it? 
Puppygirl Linux is a Arch Linux (arch btw) Linux distribution, oriented for usage by Pet Girls/boys (puppy, bunny, etc) Furries, any more!

But realistically this distro can be used by anyone. Its based on [Arch Linux](https://archlinux.org/), a Rolling-Release Linux Distribution oriented at being lightweight and flexible. Puppygirl Linux is aimed at being easier to get-up and running, General Usage, And Stability.
